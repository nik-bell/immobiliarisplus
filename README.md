# **Laboratorio Integrato – ImmobiliarisPlus (Gruppo 9 → NOVEGRUPPO)**

Progetto Didattico – ITS-ICT Piemonte – A.A. 2025/2026

## **Cos’è il Laboratorio Integrato**

Il Laboratorio Integrato è un modulo formativo dell’ITS-ICT Piemonte – Academy per le Tecnologie dell’Informazione e della Comunicazione, in cui tre percorsi professionali collaborano alla realizzazione di un progetto reale:

* **Software Developer (SWD)**
* **Web Developer (WDV)**
* **Digital Strategist (DSG)**

Si tratta di un progetto didattico e collaborativo, simile a una simulazione aziendale, sviluppato con metodologie professionali (GitHub, documentazione tecnica, presentazioni finali).

Ogni gruppo opera come un vero team di un’agenzia digitale, integrando competenze tecniche e strategiche per rispondere alle esigenze concrete di un cliente reale.

## **Mentor del Corso**

Il progetto è seguito e supervisionato dai tre mentor del Laboratorio:

* **Vincenzo Franciosa – Mentor Software Developer**
* **Shadi Lahham – Mentor Web Developer**
* **Michele Leone – Mentor Digital Strategist**


## **Azienda Cliente**

**Immobiliaris – Agenzia Immobiliare**
Referente aziendale: **Paolo Ghirlinzoni**
Settore: **Real Estate – Acquisizione di immobili**
Area operativa: **Piemonte (Torino, Cuneo, Alessandria, Asti)**

Il cliente ha richiesto la progettazione di un portale digitale ispirato al modello Gromia, focalizzato sull’acquisizione di immobili in esclusiva.


## **Requisiti Principali del Cliente**

### **Obiettivi**

* Modernizzare il processo di acquisizione immobiliare
* Raggiungere un target più giovane (35–55 anni)
* Realizzare un portale semplice, affidabile e innovativo
* Supportare la strategia di comunicazione digitale
* Impostare una campagna paid di lead generation (Meta + Google)

### **Funzionalità Richieste per il Portale**

* Onboarding dei proprietari tramite form multi-step
* Valutazione automatica dell’immobile (entro 72 ore)
* Gestione backoffice per admin e agenti
* Proposta di incarico di vendita in esclusiva
* Integrazione con strumenti di marketing automation
* Design responsive e ottimizzazione SEO


## **Tecnologie Richieste**

* **Backend:** Java + Spring Boot
* **Frontend:** HTML/SCSS/JS/TypeScript oppure React (no Bootstrap)
* **Database:** MySQL
* **Versionamento:** GitHub (repository pubblico + Projects)
* **Documentazione:** README, Swagger, JSDoc

# **Integrated Laboratory – ImmobiliarisPlus (Group 9 → NOVEGRUPPO)**

Educational Project – ITS-ICT Piemonte – A.Y. 2025/2026

## **What is the Integrated Laboratory?**

The Integrated Laboratory is a training module of ITS-ICT Piemonte – Academy for Information and Communication Technologies, where three professional tracks collaborate on the development of a real project:

* **Software Developer (SWD)**
* **Web Developer (WDV)**
* **Digital Strategist (DSG)**

It is an educational and collaborative project, similar to a real company simulation, developed using real methodologies (GitHub, technical documentation, final presentations).

Each group works as if it were a real team within a digital agency, integrating technical and strategic skills to solve a real client request.


## **Course Mentors**

The project is supervised and guided by the three mentors of the Laboratory:

* **Vincenzo Franciosa – Software Developer Mentor**
* **Shadi Lahham – Web Developer Mentor**
* **Michele Leone – Digital Strategist Mentor**


## **Client Company**

**Immobiliaris – Real Estate Agency**
Corporate contact: **Paolo Ghirlinzoni**
Industry: **Real Estate – Property Acquisition**
Area: **Piedmont (Turin, Cuneo, Alessandria, Asti)**

The client requested the design of a digital portal inspired by the Gromia model, focused on acquiring properties under exclusive contract.


## **Main Requirements Provided by the Client**

### **Objectives**

* Modernize the property acquisition process
* Reach a younger target audience (35–55 years)
* Create a simple, reliable, and innovative portal
* Support the digital communication strategy
* Set up a paid lead generation campaign (Meta + Google)

### **Required Portal Features**

* Multi-step onboarding form for property owners
* Automated property valuation (within 72 hours)
* Backoffice management for admins and agents
* Exclusive sales contract proposal
* Integration with marketing automation tools
* Responsive design and SEO


## **Technologies Required**

* **Backend:** Java + Spring Boot
* **Frontend:** HTML/SCSS/JS/TypeScript or React (no Bootstrap)
* **Database:** MySQL
* **Versioning:** GitHub (public repository + Projects)
* **Documentation:** README, Swagger, JSDoc
